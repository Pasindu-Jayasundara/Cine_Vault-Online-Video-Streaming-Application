# CineVault - Online Video Streaming Platform
> A web application which allow online movie and tv series streaming and other features based on subscription

### Features
1) Login / Register
2) Watch Movies / Tv-Series
3) Online Purchsing
4) Subscription
5) Cart
6) Favourite
7) Settings
8) Mobile Friendly design
